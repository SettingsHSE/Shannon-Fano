В этой задаче надо построить код Шеннона-Фано. Код Фано используется при сжатии данных. Символы исходного алфавита кодируются неравномерным двоичным префиксным кодом. Префиксный код – такой код, в котором ни одно кодовое слово не является началом другого кодового слова. Код Шеннона-Фано строится на основе известных вероятностей появления символов исходного алфавита в сообщении. Сумма всех вероятностей должна быть равна единице, массив вероятностей должен быть отсортирован по убыванию. (Но в нашем случае мы обойдемся сортировкой по количеству, по сути это одно и тоже)

Для построения кода Шеннона-Фано также используется дерево, как и для построения кода Хаффмана. Построение начинается с листьев дерева.

Формат ввода
В первой строке - мощность алфавита N (максимум - INT_MAX), то есть количество разных символов в сообщении. 
Во второй строке - N целых положительных чисел - количества появлений каждого символа алфавита в сообщении в порядке убывания, сумма всех символов не превышает INT_MAX.

Формат вывода
Строка с полученными коды символов, разделенными пробелами, расставлены в порядке убывания (по количеству повторений).
